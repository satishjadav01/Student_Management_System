# Student Management System (Python)

A command-line based Student Management System built using pure Python.  
The project performs basic CRUD operations with persistent local storage.

## Features
- Add new students
- View all students
- Delete student records
- Search students by ID
- Data stored locally using JSON
- cant not store dublicate records 

## Technologies Used
- Python
- JSON for data persistence

